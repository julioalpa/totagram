# totagram

Generar entorno virtual

  - python3 -m venv .env
 
Activarlo

  - source ./env/bin/activate

Instalar dependencias en el entorno

  - pip install django
  - pip install Pillow
  
Migrar DB

  - python3 manage.py makemigrations
  - python3 manage.py migrate
  
Correr Server

  - python3 manage.py runserver
  
url -> http://localhost:8000
